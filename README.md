# AIDE-edu

AIDE-edu는 **교육·연구용 LLM 실험 플랫폼**입니다.  
웹 브라우저에서 바로 다음 작업을 할 수 있습니다.

- 채팅 / 프롬프트 실험
- JSON/JSONL 데이터셋 업로드
- SFT 기반 모델 학습 (Full Fine-Tuning / LoRA)
- 학습된 모델 평가 (Evaluation)
- CPU / GPU 모두 지원 (Docker 기반)
